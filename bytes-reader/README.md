# Bytes Reader
Application reads 20 bytes from last -50 position text.   

Example:

```
LETTER IS ASSIGNED A VALUE ACCORDING TO ITS POSITION WITHIN THE ASCII TABLE.
EVERY LETTER, NUMBER, PUNCTUATION(35) MARK, ETC. IS GIVEN A UNIQUE CODE.
```

Gives:

```
PUNCTUATION(35) MARK
```
